## 依赖要求

-   node >=18

## 安装

```sh
npm install commitlint-config-cmyr
```

## 使用

```sh
npm run start
```

## 开发

```sh
npm run dev
```

## 编译

```sh
npm run build
```

## Lint

```sh
npm run lint
```

## Commit

```sh
npm run commit
```
